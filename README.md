# AzureOpenAI_Chat
Azure OpenAI チャット Webアプリ。 開発言語：Python。GUI:Streamlit。 

アプリの名前：NyAzuChat（ニャアズチャット）。  

#準備すること：・AzureポータルでAzure OpenAIサービスを設定する。 ・ローカル開発環境で.envファイルを利用する場合、.envファイルをプロジェクトのルートディレクトリ（app.pyと同じ場所）に配置し設定する。  

#料金が発生する状況の詳細： Azure OpenAI Service は無料プランの提供がありません。このため、料金が発生します。30日間の無料クレジットをお持ちの場合、ご利用料金はそのクレジットから補填されるため、お支払いなしで使用できます。
